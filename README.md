# 난임 치료 성공 예측 모델

## 프로젝트 개요
불임 치료 데이터를 활용한 임신 성공 여부 예측 모델 개발. 데이터 전처리, 파생 변수 생성, 하이퍼파라미터 튜닝 및 앙상블을 통해 예측 성능(ROC-AUC) 최적화.

## 주요 전처리 및 피처 엔지니어링
- 데이터 정제: 문자열 제거 및 결측치 처리 (-1, Unknown)
- 인코딩: 연령대별 가중치 매핑 (0~5)
- 파생 변수: 
    - 불임_원인_개수: 원인 복합성 수치화
    - 난자_배아_전환율: 시술 효율성 지표
    - 배아_활용도: 자원 관리 지표
